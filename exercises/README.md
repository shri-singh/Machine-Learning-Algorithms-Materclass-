# Exercises

Practice notebooks for each module. Each notebook contains exercises of increasing difficulty with solutions provided in collapsible sections.

| File | Covers |
|------|--------|
| `ML100_exercises.ipynb` | Data Splitting, Feature Engineering, Pipelines |
| `ML200_exercises.ipynb` | Linear Regression, Regularization, Metrics |
| `ML300_exercises.ipynb` | Logistic Regression, Classification Metrics, Imbalance |
| `ML400_exercises.ipynb` | KNN, KMeans, Hierarchical, DBSCAN, PCA |
| `ML500_exercises.ipynb` | Decision Trees, Random Forest, Boosting |
| `ML600_exercises.ipynb` | Optimization, Feature Selection, End-to-End ML |
